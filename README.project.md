# Project Name

This project was created using the **Next.js Docker Development Template**.

It includes a fully configured local development environment running entirely inside Docker.

---

# Development Environment

The stack includes:

* Next.js
* Docker
* PostgreSQL
* Redis
* pgAdmin
* Redis Commander

All services run inside Docker containers so **no database or cache services need to be installed on your system**.

---

# Start Development

Start the development environment:

```bash
./dev.sh start
```

---

# Rebuild Containers

If dependencies change or containers fail:

```bash
./dev.sh rebuild
```

---

# Stop the Environment

Stop all running containers:

```bash
./dev.sh stop
```

---

# View Logs

Follow container logs in real time:

```bash
./dev.sh logs
```

---

# Services

Once the environment is running you can access the following services.

### Web Application

```
http://localhost:3000
```

### pgAdmin

Database management interface.

```
http://localhost:5050
```

### Redis Commander

Redis management interface.

```
http://localhost:8081
```

---

# Environment Variables

Project configuration is stored in:

```
.env
```

This file is automatically generated during project setup.

---

# Reset the Environment

If containers break or the database needs resetting:

```bash
./dev.sh reset
```

Then rebuild the environment:

```bash
./dev.sh rebuild
```
